# 📝 Todo List App

A simple and responsive **Todo List Web Application** built to manage daily tasks efficiently. This app allows users to add, delete, and track tasks with a clean and user-friendly interface.

## 📌 Features

* ✅ Add new tasks
* 🗑️ Delete tasks
* ✔️ Mark tasks as completed
* 💾 Data stored using **Local Storage**
* 📱 Fully responsive design
* ⚡ Fast and lightweight

## ⚙️ How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/RambihariPatel/TodoList.git
```

2. Open folder

```bash
cd TodoList
```

3. Run the project

* Simply open `index.html` in your browser

---

## 💡 How It Works

* User enters a task and clicks **Add**
* Task is stored in **Local Storage**
* Tasks are displayed on screen
* User can mark tasks as completed or delete them
